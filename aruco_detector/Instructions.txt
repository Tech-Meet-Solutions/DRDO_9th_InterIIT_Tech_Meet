Place "aruco_marker.world" in ardupilot_gazebo/worlds

roscd gazebo_ros
cd launch
sudo cp empty_world.launch iris_world.launch
sudo gedit iris_world.launch
Change the line containing empty_world.world to aruco_marker.world
Save
Close iris_world.launch

Place the package aruco_detector in src

Run catkin_make

The run the following:
In the directory of start2.sh and start.sh:
chmod +x start2.sh
chmod +x start.sh

./start2.sh
./start.sh

Navigate to the directory containing detectAruco.py and run:

chmod +x detectAruco.py

Open a new terminal and run:
rosrun aruco_detector detectAruco.py
